🎓 College DBMS Project

A Database Management System (DBMS) mini project created as part of a college academic assignment.
This project demonstrates how relational databases are designed and how structured data can be stored, managed, and retrieved efficiently using SQL queries.
The project focuses on applying core DBMS concepts such as table relationships, normalization, constraints, and query execution.

📌 Project Overview
Database systems are used everywhere — from banking systems to social media platforms.
This project demonstrates the fundamental working of a Relational Database Management System (RDBMS).

🚀 Features

✨ Solve SQL question with unique frontend interface
✨ Perform data operations using SQL
✨ SQL queries for data retrieval
✨ Data insertion, update, and deletion
✨ Demonstration of real-world database usage

🛠 Technologies Used
Technology	Purpose

🧠 DBMS Concepts Implemented

This project includes implementation of several important database concepts:

📊 Entity Relationship Model (ER Model)
🗂 Database Table Creation (DDL)
✏ Data Manipulation (INSERT, UPDATE, DELETE)
🔎 Data Retrieval using SELECT Queries
🔑 Primary Keys & Foreign Keys
🔗 Table Relationships
📉 Database Normalization
